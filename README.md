## 📝 **Sobre o Curso**

Este repositório foi criado para te guiar pelos conceitos básicos do C# com foco na prática. Ao final, você estará preparado para desenvolver seus próprios projetos!

### 📌 **O que você vai aprender:**
- 📋 Estrutura básica de um programa C#
- ✏️ Variáveis, tipos de dados e operadores
- 🔄 Estruturas de controle: `if`, `else`, `switch`, `loops` e muito mais!
- 🛠️ Métodos e funções
- 🧑‍💻 Programação Orientada a Objetos (POO)
- 📂 Manipulação de arquivos
- 🐞 Debugging e tratamento de erros

## ⚙️ **Pré-requisitos**

Antes de começar, certifique-se de ter:

- [Visual Studio Code](https://code.visualstudio.com/) ou [Visual Studio](https://visualstudio.microsoft.com/)
- [.NET SDK](https://dotnet.microsoft.com/download) (versão mais recente)
- Conhecimentos básicos de lógica de programação
